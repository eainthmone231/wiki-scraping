# wiki-scraping
Aim for burmese text data collection from wikpedia

## How to run
**Link Scrape** <br>
   python link_scrape.py <**link**> <**save_file_name**>

**Content Scrape** <br>
   python content_scrape.py <**link**> <**save_file_name**>

**Scrape all links within a given json** <br>
    python scrape_by_json.py
